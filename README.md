# Насенников Максим 501497 индивидуальные правки 
## добавление модуля gui_app
### В процессе доработки проекта был добавлен новый модуль gui_app, который отвечает за графический интерфейс приложения. Теперь пользователь может работать с программой через понятное и удобное окно, не используя консоль. В интерфейсе предусмотрены наглядные элементы управления и удобные кнопки, на которые можно нажимать для выполнения нужных действий. Модуль gui_app объединяет основной функционал приложения, обрабатывает действия пользователя и отображает результаты работы программы.

В связи с переходом на графический интерфейс модули main.py и interface.py были удалены, так как они больше не нужны. Ранее они использовались для запуска программы и взаимодействия через консоль, однако после внедрения GUI их функциональность стала избыточной.
